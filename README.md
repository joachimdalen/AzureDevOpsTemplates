| File                                               | Description                                                             |
| -------------------------------------------------- | ----------------------------------------------------------------------- |
| [tag-sources.yml](./templates/git/tag-sources.yml) | Create a git tag and push it                                            |
| [doctl.yml](./templates/tools/doctl.yml)           | Install and authenticate [doctl](https://github.com/digitalocean/doctl) |
